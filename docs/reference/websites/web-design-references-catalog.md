# 웹 디자인·인터랙티브 레퍼런스 카탈로그 (다양화)

> Messenger(abeto)에서 출발하되 **특정 방향에 묶이지 않고** 인터랙티브 웹 전 영역에서 폭넓게 뽑은 레퍼런스 라이브러리. 카테고리별로 계속 추가하는 성장형 문서.
> ⚠️ 다수 항목은 봇 차단·JS 렌더로 본문 자동 추출이 안 돼 어워드(Awwwards/FWA)·공식 케이스스터디·리뷰 기반. 직접 방문 시 체감 갱신.
