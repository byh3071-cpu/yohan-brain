# 2026-06-12 — 별자리 탭에 옵시디언식 2D 그래프뷰 추가 (2D/3D 모드 토글)

## 결정

1. **별자리 탭을 2D/3D 듀얼 모드로 확장** — 기존 3D 은하 뷰(`constellation-view.tsx`)는 그대로 유지하고, 옵시디언 그래프뷰를 충실히 카피한 2D 뷰(`graph-view-2d.tsx`)를 추가. 기본 모드는 **2D**(사용자 요청 취지), 툴바 토글로 전환.
2. **포스 시뮬레이션은 자체 구현** — `lib/force-sim-2d.ts`. d3-force 동작(링크 스프링+차수 bias·다체 반발 1/d·중심 인력·alpha 감쇠/재가열)을 의존성 없이 재현. 패키지 추가 없음.
3. **옵시디언 설정 패널 4섹션 동일 구성** — 필터(검색·고아 노드 표시) / 그룹(카테고리 색상·표시 토글 = 옵시디언 Groups 대응) / 표시(화살표·텍스트 흐림 임계값·노드 크기·선 굵기) / 힘(중심 0~1·반발 0~20·링크 0~1·거리 30~500 — 옵시디언 슬라이더 범위 동일).
4. **인터랙션 동일** — 노드 드래그(시뮬레이션 재가열), 배경 팬, 휠 줌(커서 기준), 호버 시 이웃 하이라이트+나머지 페이드, 클릭 → 문서 미리보기, 더블클릭/버튼 → 화면 자동 맞춤.
5. **기존 데이터 파이프라인 재사용** — `ConstellationData`(노드·엣지·D-2 시점 필터)를 그대로 받아 2D에서도 시점 슬라이더 동작. 노드 증감 시 기존 좌표·속도 보존 + 신규 노드는 3D 클러스터 좌표(x,z) 시드로 출발.

## 근거

- 사용자 요청: "지식 별자리(옵시디언 그래프뷰를 본딴 것)를 완전히 카피해 달라." 옵시디언 그래프뷰의 본질은 2D 포스 다이렉티드 + 설정 패널이므로 3D 개조가 아닌 별도 2D 모드가 충실한 카피.
- 3D 뷰는 SPEC D-2(시점 슬라이더)·D-4(성운) 등 스펙 반영 자산이라 제거하지 않고 병행 유지.
- d3-force 패키지 미도입: ~600노드 O(n²) 자체 틱으로 충분, 의존성·번들 증가 회피.
