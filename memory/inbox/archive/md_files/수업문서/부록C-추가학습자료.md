이 부록은 교재를 마친 뒤 더 깊이 공부하고 싶은 분들을 위한 추가 학습 자료 모음이에요. 난이도별로 정리했으니 자기 수준에 맞는 것부터 시작하면 돼요.

난이도 표기: ★ = 초급(배경지식 없이 시작 가능), ★★ = 중급(이 교재를 마친 수준), ★★★ = 고급(기술적 배경 필요)

# 1. 공식 문서
// ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

각 AI 서비스 제공사가 직접 작성한 공식 문서예요. 가장 정확하고 최신 정보를 담고 있으니, 특정 도구를 깊이 사용하려면 공식 문서부터 읽는 것이 좋아요.

**Anthropic Docs** (★~★★)\
#link("https://docs.anthropic.com")\
Claude의 공식 문서예요. 프롬프트 엔지니어링 가이드, API 레퍼런스, 모델 카드를 포함해요. 이 교재 이론 3차시와 직접 연결돼요.

**Claude Code Docs** (★★)\
#link("https://docs.anthropic.com/en/docs/claude-code")\
Claude Code의 설치, 설정, CLAUDE.md, MCP, Skills 등 전체 기능을 다루는 공식 문서예요. 실습 7~12차시의 원본 소스에 해당해요.

**OpenAI Docs** (★~★★)\
#link("https://platform.openai.com/docs")\
GPT 시리즈의 공식 문서예요. API 사용법, 프롬프트 가이드, 함수 호출, 어시스턴트 API를 다뤄요.

**OpenAI Prompt Engineering Guide** (★)\
#link("https://platform.openai.com/docs/guides/prompt-engineering")\
OpenAI가 제공하는 프롬프트 엔지니어링 모범 사례 가이드예요. 6가지 전략으로 구성돼 있어요.

**Google AI for Developers** (★★)\
#link("https://ai.google.dev")\
Gemini API의 공식 문서예요. 멀티모달 입력, 구조화 출력, 함수 호출을 다뤄요.

**Hugging Face Docs** (★★~★★★)\
#link("https://huggingface.co/docs")\
오픈소스 모델 허브이자 Transformers 라이브러리의 공식 문서예요. 오픈 웨이트 모델을 직접 다루고 싶을 때 필수예요.

**MCP Specification** (★★★)\
#link("https://modelcontextprotocol.io")\
Model Context Protocol의 공식 사양 문서예요. MCP 서버 개발과 통합에 관한 기술 명세를 담고 있어요.

# 2. 무료 강의
// ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

체계적으로 배우고 싶다면 온라인 강의가 좋아요. 아래 강의들은 모두 무료로 수강할 수 있어요.

**Google AI Essentials** (★)\
#link("https://grow.google/ai-essentials")\
Google이 Coursera에서 제공하는 AI 기초 과정이에요. 비기술직도 수강할 수 있어요.

**DeepLearning.AI Short Courses** (★~★★)\
#link("https://deeplearning.ai/short-courses")\
Andrew Ng의 DeepLearning.AI에서 제공하는 단기 과정 모음이에요. 주제별 1~2시간 과정이에요.

**Anthropic Prompt Engineering Course** (★★)\
#link("https://docs.anthropic.com/en/docs/build-with-claude/prompt-engineering/overview")\
Anthropic이 직접 만든 프롬프트 엔지니어링 과정이에요. 이 교재 이론 3차시를 더 깊이 다뤄요.

**CS50's Introduction to AI** (★★)\
#link("https://cs50.harvard.edu/ai")\
하버드 대학교의 AI 입문 과정이에요. 탐색, 지식 표현, 머신러닝, 신경망의 기초를 다뤄요.

**Stanford CS229: Machine Learning** (★★★)\
#link("https://cs229.stanford.edu")\
스탠포드 대학교의 머신러닝 과정이에요. 수학적 기초부터 다뤄서 이론을 깊이 이해하고 싶은 분에게 적합해요.

**fast.ai Practical Deep Learning** (★★)\
#link("https://course.fast.ai")\
코드 중심으로 딥러닝을 배우는 실용적 과정이에요. Python 기초만 있으면 시작할 수 있어요.

**Hugging Face NLP Course** (★★~★★★)\
#link("https://huggingface.co/learn/nlp-course")\
Hugging Face가 제공하는 NLP 과정이에요. Transformer 아키텍처, 파인튜닝, 모델 배포를 직접 실습해요.

# 3. 한국어 자료
// ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

한국어로 AI를 공부할 수 있는 자료들이에요. 영어 원문을 읽기 부담스러울 때 좋은 출발점이에요.

**모두의 연구소** (★~★★)\
#link("https://modulabs.co.kr")\
AI/ML 학습 커뮤니티예요. 논문 읽기 모임, 프로젝트 기반 스터디, 세미나를 운영해요.

**네이버 부스트코스 AI** (★~★★)\
#link("https://boostcourse.org")\
네이버가 제공하는 무료 AI 교육 플랫폼이에요. 파이썬 기초부터 딥러닝까지 한국어로 체계적으로 배울 수 있어요.

**테디노트 유튜브** (★★)\
#link("https://youtube.com/@teddynote")\
AI/LLM 관련 한국어 유튜브 채널이에요. LangChain, RAG, 프롬프트 엔지니어링 등 실습 중심 튜토리얼을 제공해요.

**노마드코더 AI 콘텐츠** (★~★★)\
#link("https://nomadcoders.co")\
프로그래밍 교육 플랫폼의 AI 관련 무료 콘텐츠예요. ChatGPT API 활용, AI 앱 개발 등을 다뤄요.

**AI타임스** (★)\
#link("https://aitimes.com")\
AI 전문 뉴스 매체예요. 국내외 AI 산업 동향, 정책, 기술 트렌드를 한국어로 전해요.

**Google ML Crash Course 한국어** (★★)\
#link("https://developers.google.com/machine-learning/crash-course")\
Google의 ML 속성 과정 한국어 버전이에요. 머신러닝 기초 개념을 시각적으로 설명해요.

**위키독스 -- 딥러닝을 이용한 자연어 처리 입문** (★★)\
#link("https://wikidocs.net/book/2155")\
한국어로 된 NLP 입문서예요. 토큰화, 임베딩, 트랜스포머까지 기초부터 차근차근 설명해요.

# 4. 논문/깊은 읽기
// ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

AI의 원리를 깊이 이해하고 싶다면 원논문을 읽어보세요. 아래는 교재에서 언급되거나 배경이 된 핵심 논문들이에요. 논문 읽기가 어렵다면 요약본이나 블로그 해설부터 시작해도 좋아요.

**Vaswani et al. (2017) -- "Attention Is All You Need"** (★★★)\
트랜스포머 아키텍처를 처음 제안한 논문이에요. 현대 모든 LLM의 기반 구조를 만든 역사적 논문이에요.

**Brown et al. (2020) -- "Language Models are Few-Shot Learners" (GPT-3)** (★★★)\
프롬프트 안의 예시만으로 다양한 과제를 수행하는 인컨텍스트 학습을 보인 논문이에요. 프롬프트 엔지니어링의 출발점이에요.

**Kaplan et al. (2020) -- "Scaling Laws for Neural Language Models"** (★★★)\
모델 크기, 데이터, 연산량과 성능의 관계를 정량적으로 보인 논문이에요. 이론 1차시 스케일링 법칙의 원본이에요.

**Wei et al. (2022) -- "Chain-of-Thought Prompting Elicits Reasoning"** (★★★)\
CoT(사고의 연쇄) 프롬프팅을 제안한 논문이에요. 부록 B의 E-1 템플릿의 원본이에요.

**Wang et al. (2022) -- "Self-Consistency Improves Chain of Thought Reasoning"** (★★★)\
Self-Consistency 기법을 제안한 논문이에요. 부록 B의 E-2 템플릿의 원본이에요.

**Yao et al. (2023) -- "Tree of Thoughts: Deliberate Problem Solving with LLMs"** (★★★)\
Tree of Thoughts를 제안한 논문이에요. 부록 B의 E-3 템플릿의 원본이에요.

**Suzgun & Kalai (2024) -- "Meta-Prompting: Task-Agnostic Scaffolding"** (★★★)\
메타 프롬프팅의 Conductor-Expert 패턴을 정식화한 논문이에요. 부록 B의 E-4 템플릿의 원본이에요.

**Dell'Acqua et al. (2023) -- "Navigating the Jagged Technological Frontier"** (★★)\
AI 능력의 들쭉날쭉한 경계(Jagged Frontier)를 실증한 하버드/BCG 공동 연구예요. 이론 4차시에서 다뤘어요.

**Ouyang et al. (2022) -- "Training LMs to Follow Instructions with Human Feedback" (InstructGPT)** (★★★)\
RLHF를 본격 적용하여 지시 따르기 능력을 개선한 논문이에요. 이론 2차시 정렬의 핵심 배경이에요.

**Mollick (2024) -- "Good Enough Prompting"** (★)\
프롬프트 엔지니어링의 실용적 원칙을 정리한 글이에요. "10시간 사용하면 실력이 는다"는 메시지가 핵심이에요.

# 5. 도구/실습 환경
// ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

배운 것을 직접 실습할 수 있는 도구와 환경이에요. 교재 실습에서 사용한 도구들과 추가로 탐색해볼 만한 도구들을 모았어요.

**ChatGPT** (★)\
#link("https://chatgpt.com")\
OpenAI의 대화형 AI 인터페이스예요. 무료 버전으로 기본 프롬프트 연습을 시작하기에 가장 접근성이 높아요.

**Claude** (★)\
#link("https://claude.ai")\
Anthropic의 대화형 AI 인터페이스예요. 긴 문서 분석, 구조화된 출력에 강점이 있어요.

**Claude Code** (★★)\
#link("https://docs.anthropic.com/en/docs/claude-code")\
Claude의 터미널 기반 에이전트 도구예요. 파일 읽기/쓰기, 코드 실행, MCP, Skills를 지원해요.

**Google AI Studio** (★)\
#link("https://aistudio.google.com")\
Google의 Gemini 모델을 실험할 수 있는 웹 인터페이스예요. 멀티모달 입력, 프롬프트 테스트에 활용할 수 있어요.

**Hugging Face Spaces** (★~★★)\
#link("https://huggingface.co/spaces")\
오픈소스 AI 모델들을 웹에서 바로 실험할 수 있는 환경이에요. 설치 없이 다양한 모델을 체험해볼 수 있어요.

**Google Colab** (★★)\
#link("https://colab.research.google.com")\
무료 GPU가 제공되는 클라우드 Jupyter 노트북이에요. Python 코드로 AI 모델을 직접 다뤄보고 싶을 때 좋아요.

**LM Studio** (★★)\
#link("https://lmstudio.ai")\
로컬 컴퓨터에서 오픈 웨이트 LLM을 실행할 수 있는 데스크톱 앱이에요. 클라우드 없이 AI를 실험하고 싶을 때 유용해요.

**Ollama** (★★)\
#link("https://ollama.com")\
커맨드라인에서 오픈 웨이트 모델을 간단히 실행하는 도구예요. 로컬 AI 서버를 만들 수 있어요.

**vLLM** (★★★)\
#link("https://docs.vllm.ai")\
오픈 웨이트 모델을 프로덕션 수준으로 서빙하는 고성능 추론 엔진이에요. 이론 1차시에서 언급했어요.

**LangChain** (★★★)\
#link("https://langchain.com")\
LLM 기반 애플리케이션을 개발하는 프레임워크예요. RAG, 에이전트, 도구 연동을 코드로 구현할 수 있어요.
