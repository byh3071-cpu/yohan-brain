## AI에게 데이터를 넘기는 법

AI에게 데이터를 넘기는 가장 간단한 방법은 **텍스트로 붙여넣기**예요. 엑셀에서 데이터를 복사하면 탭으로 구분된 텍스트가 되거든요. 이걸 그대로 대화창에 붙여넣으면 돼요.

데이터를 넘길 때 알아둘 점 세 가지가 있어요.

#table(
  columns: (auto, 1fr),
  fill: (x, y) => if y == 0 { practice-color.primary } else if calc.rem(y, 2) == 1 { practice-color.bg } else { color.white },
  stroke: 0.5pt + practice-color.border,
  inset: 10pt,
  table.header(
    text(fill: color.white, weight: "bold")[원칙],
    text(fill: color.white, weight: "bold")[설명],
  ),
  text(weight: "bold", fill: practice-color.primary)[열 이름을 명확히], [A열, B열이 아니라 "월", "매출", "고객수" 같은 이름이 있어야 AI가 의미를 이해해요],
  text(weight: "bold", fill: practice-color.primary)[단위를 표시해줘야 해요], [숫자만 있으면 AI가 만원인지 원인지 %인지 모를 수 있어요. "매출(만원)" 같이 표시하세요],
  text(weight: "bold", fill: practice-color.primary)[데이터 양에 한계가 있어요], [무료 버전은 한 번에 넘길 수 있는 텍스트 양에 제한이 있어요. 100행 이내가 적당해요],
)

# 실습 시작: 매출 데이터 분석
// ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

#practice-step(1, "데이터 붙여넣고 분석 요청하기")[
  아래 실습용 데이터를 복사해서 AI에 붙여넣으세요. 본인의 실제 업무 데이터가 있다면 그것을 사용해도 좋아요. (민감한 데이터는 가명 처리 후 사용하세요.)

  ```
  월,매출(만원),신규고객(명),이탈고객(명),마케팅비(만원),NPS점수
  2025-07,3200,120,15,450,72
  2025-08,3100,110,22,480,68
  2025-09,3400,135,18,520,71
  2025-10,3800,160,12,600,75
  2025-11,4200,180,10,650,78
  2025-12,4500,195,8,700,82
  2026-01,4100,170,25,680,74
  2026-02,3900,155,30,620,70
  2026-03,4300,185,14,710,79
  ```

  이 데이터를 붙여넣고 아래 프롬프트를 추가하세요.

  ```
  위 데이터를 분석해줘.

  분석 요청:
  1. 전체 추세 요약 (3줄)
  2. 눈에 띄는 이상치(anomaly)가 있다면 지적
  3. 매출과 가장 상관관계가 높아 보이는 변수는?
  4. 데이터에서 읽을 수 있는 경고 신호가 있다면?

  주의:
  - 데이터에 있는 숫자만 사용해
  - 추측이 필요한 부분은 "(추측)"으로 표시해
  - 인과관계와 상관관계를 구분해서 말해
  ```

  결과를 확인해보세요. AI가 데이터에서 패턴을 찾아냈나요?

  아마 이런 분석이 나올 거예요:
  - 매출은 전반적 상승 추세이지만 2026년 1~2월에 하락했어요
  - 2026년 1~2월에 이탈 고객이 급증했어요 (25명, 30명)
  - 매출과 신규 고객 수의 상관관계가 높아 보여요
  - 2월의 NPS 점수 하락(70점)은 경고 신호예요

  > ***핵심****: AI가 돌려준 분석에서 "상관관계"와 "인과관계"를 구분하는 게 중요해요. "마케팅비를 늘리면 매출이 는다"는 상관관계일 수 있지만 인과관계는 아닐 수 있어요. AI는 이 구분을 놓치는 경우가 있으니 사람이 판단해야 해요.
]

#practice-step(2, "차트 설명 요청하기")[
  무료 AI는 대부분 차트를 직접 그려주지 못해요. 하지만 **차트 설명**은 해줄 수 있어요. 데이터를 어떤 차트로 시각화하면 좋을지, 그리고 그 차트에서 무엇이 보일지를 물어볼게요.

  같은 대화에서 아래 프롬프트를 이어서 입력하세요.

  ```
  위 데이터를 시각화한다면 어떤 차트를 쓰는 게 좋을지 추천해줘.

  요청:
  1. 차트 3개를 추천해줘 (종류, 무엇을 보여주는지, 축 설정)
  2. 각 차트에서 핵심적으로 보이는 패턴을 설명해줘
  3. 경영진 보고용으로 가장 효과적인 차트 1개를 고르고 이유를 설명해줘

  형식: 번호 목록
  ```

  AI가 추천하는 차트 종류를 확인해보세요. 보통 이런 추천이 나와요:
  - 매출 추이 -> 꺾은선 그래프 (시간 축)
  - 채널별 비교 -> 막대 그래프
  - 매출 vs 마케팅비 -> 산점도

  AI가 "이 차트에서 이런 패턴이 보인다"고 설명하는 부분이 핵심이에요. 실제로 차트를 만들기 전에 AI의 설명으로 **무엇을 보여줄 것인지**를 먼저 결정할 수 있거든요.

  #try-it[
    AI가 추천한 차트 중 하나를 실제로 만들어보세요.
    - 엑셀/구글 시트에 데이터를 넣고 차트를 생성해요
    - AI의 설명과 실제 차트를 비교해보세요
    - AI의 패턴 분석이 맞았나요?
  ]
]

#practice-step(3, "인사이트 추출하기")[
  데이터를 분석했으니, 이제 **의사결정에 필요한 인사이트**를 뽑아볼게요. 같은 대화에서 이어서 입력하세요.

  ```
  지금까지의 분석을 바탕으로, 아래 질문에 답해줘.

  1. 2026년 2분기에 가장 주의해야 할 지표는 무엇이고, 왜?
  2. 마케팅 예산 배분을 조정한다면 어떻게 바꿀 것을 제안해?
     (단, 총 예산은 현재와 동일하다는 전제)
  3. 이탈 고객 급증(1~2월)의 가능한 원인 3가지를 추측해봐.
     (이건 데이터만으로 알 수 없으니 "(추측)"이라고 명시해)

  형식:
  - 각 답변에 근거가 되는 수치를 반드시 포함
  - 확실한 것과 추측인 것을 명확히 구분
  ```

  결과에서 "확실한 것"과 "추측인 것"이 구분돼 있는지 확인하세요.

  AI가 내놓는 인사이트의 가치는 **새로운 관점**에 있어요. 사람이 데이터를 보면 익숙한 패턴만 보게 되는 경향이 있어요. AI는 다른 각도에서 패턴을 제시해줘요. 물론 그 해석이 맞는지는 사람이 판단해야 해요.

  > ****핵심****: AI 데이터 분석의 올바른 활용: **AI가 패턴을 발견하고, 사람이 의미를 판단하는 것**이에요. AI의 분석을 맹신하지 말고, "이 해석이 맞나?"라는 질문을 항상 달고 다니세요.
]

#practice-step(4, "보고서 초안 작성하기")[
  마지막으로, 지금까지의 분석을 하나의 보고서로 묶어볼게요. 같은 대화에서 이어서 입력하세요.

  ```
  지금까지 분석한 내용을 바탕으로
  "2026년 2분기 마케팅 전략 제안" 보고서 초안을 작성해줘.

  보고서 구조:
  1. 1분기 성과 요약 (핵심 수치 3개)
  2. 주요 발견사항 (긍정 2개 + 경고 2개)
  3. 2분기 전략 제안 (3가지, 각각 근거 수치 포함)
  4. 모니터링 지표 (2분기에 주시할 KPI 3개)

  독자: 마케팅 팀장
  톤: 데이터 중심, 간결
  분량: A4 1장 (500자 내외)
  ```

  보고서 초안이 나오면 아래 체크리스트로 검토하세요.

    text(weight: "bold", fill: practice-color.primary)[인과관계를 과도하게 주장하고 있나요?], ["상관관계"를 "원인"으로 바꿔 쓰진 않았는지 확인],

이 워크플로우에서 AI는 "속도"를 제공하고, 사람은 "판단"을 제공해요. 이 역할 분담을 기억하세요.
