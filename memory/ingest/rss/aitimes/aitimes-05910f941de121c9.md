# 프랑스, 굴 양식장에 태양광 띄운다

프랑스 남부 타우 석호에서 태양광 발전과 굴 양식을 결합한 부유식 실증 플랫폼이 시험에 들어간다. 프랑스 타우 유역 공동협의체(SMBT)는 지난 6일 솔라린토(SolarinThau) 실증 플랫폼이 7일 세트 항구를 출발해 토 석호 내 조개 양식 구역으로 이동한다고 밝혔다.플랫폼은 길이 53m, 폭 18m 규모의 부유식 조개 양식 구조물이다. 기존 양식 시설 인근에서 태양광 발전과 굴 양식이 같은 공간에서 가능한지 확인하는 데 쓰인다. 이동 과정에서는 세트 시내 주요 교량 5곳을 통과했다.실증은 2026년 7월부터 2027년 12월까
