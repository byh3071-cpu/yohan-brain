# [AI&빅데이터쇼] 마드라스체크 “AI 협업툴 ‘플로우’로 조직·개인 성과 확인”

마드라스체크(대표 이학준)는 10~12일 서울 코엑스에서 열린 ‘AI&빅데이터쇼’에 참가해 업무 협업 툴 ‘플로우’를 선보였다. 플로우는 프로젝트 협업, 업무 관리, 메신저 등을 하나로 통합한 올인원 협업 도구다. 여기에 AI 기능을 더하며 기업 데이터 맞춤 질의응답까지 가능해졌다.관계자는 “월 구독은 인당 만원 대로, 연간 구독을 할 경우 만원 이하로 할인된다”라며 “AI 기능을 같이 활용할 경우 1인당 월 2만원 대에 활용할 수 있다”라고 말했다.이처럼 챗GPT와 클로드, 제미나이 등 최신 모델을 저렴하고 편리하게 사용할 수 있
