# 퍼플렉시티, '딥 리서치'를 '컴퓨터' 플랫폼에  통합..."성능 2배 향상"

퍼플렉시티가 '딥 리서치(Deep Research)'를 자체 멀티모델 플랫폼 '컴퓨터(Computer)'에 통합하는 등 대규모 업그레이드를 단행했다. 퍼플렉시티는 11일(현지시간) 업그레이드된 딥 리서치 기능이 20개가 넘는 최신 AI 모델을 하나의 워크플로우로 연결해 운영된다고 밝혔다.이 시스템은 사용자의 복잡한 질의를 여러 세부 과제로 나눈 뒤 적합한 모델에 각각 배정해 조사·분석을 수행하고, 수백개 웹사이트의 원문 자료를 교차 검증해 출처가 명시된 심층 보고서를 자동으로 작성한다. 이번 기능은 지난 2월 공개된 퍼플렉시티 컴퓨
