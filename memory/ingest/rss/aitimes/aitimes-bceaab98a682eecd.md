# 토지 부담 줄이는 인도, 물 위 태양광으로 102GW 시장 연다

인도 정부가 수상태양광 잠재량 평가 보고서를 공개하고, 저수지와 호수 등을 활용한 개발 잠재량을 102기가와트피크(GWp)로 평가했다. 태양광 보급 확대 과정에서 토지 확보 부담이 커지면서 수상태양광을 육상 태양광 보완하려는 흐름이다.인도 정부는 10일(현지시간) 수상태양광 잠재량 평가 결과를 공식 발표했다. 보고서는 앞서 2월 이해관계자 협의회에서 초안 형태로 논의됐으며, 이번 발표를 통해 인도의 수상태양광 개발 잠재량은 102.18GWp 이상으로 제시됐다. 수상태양광은 수면 위 부유식 구조물에 태양광 모듈을 설치하는 방식이다.
