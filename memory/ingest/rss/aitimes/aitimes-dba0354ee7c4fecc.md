# 문샷 AI, ‘키미 워크’ 공개…“300개 에이전트 스웜으로 내 PC 문서·웹 조작 자동화”

문샷 AI가 사용자의 PC에서 직접 실행하는 데스크톱 AI 에이전트 ‘키미 워크(Kimi Work)’를 공개했다. 오픈클로 등의 방식과 비슷하지만, 일반 사무직도 쉽게 사용하도록 데스크톱 앱으로 패키징하고 대규모 병렬 처리를 구현한 점이 차이다문샷 AI는 8일(현지시간) 맥OS와 윈도우용 로컬 에이전트 '키미 워크'를 출시하고 다운로드를 공개했다. 로컬 파일 접근과 실제 브라우저 제어, 예약 작업 실행 기능을 결합한 지식 노동자와 개발자용 에이전트다.이 제품은 기존 오픈소스 개발자 중심의 복잡한 설치 과정을 거쳐야 했던 로컬 에이전
