# 샤오미, 오픈소스 코딩 에이전트 ‘미모 코드’ 공개…“장기 작업서 클로드 코드 능가”

샤오미가 최근 상승세를 바탕으로 새로운 AI 코딩 에이전트를 공개하며 AI 소프트웨어 개발 도구 시장에 합류했다.샤오미는 10일(현지시간) 터미널 기반 AI 코딩 에이전트 ‘미모 코드(MiMo Code)’를 오픈소스로 공개했다.미모 코드는 오픈소스 프로젝트 오픈코드(OpenCode)를 기반으로 개발한 터미널 네이티브 AI 코딩 에이전트다. 깃허브를 통해 MIT 라이선스로 공개됐다.수십~수백 단계에 이르는 장기 자동화 프로그래밍 작업에서 의사결정 품질과 작업 상태를 지속적으로 유지하는 데 초점을 맞췄다. 자체 벤치마크와 개발자 평가
