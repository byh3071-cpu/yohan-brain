# Opus 4.8 등장: 클로드는 빼앗긴 주도권을 찾아올까?

Opus 4.8이 나왔습니다. 4.7이 나온지 42일만이죠. 전에 없던 빠른 업그레이드입니다. GPT-5.5와 Codex에 주도권을 내준 한 달, 클로드의 반격 턴이죠. 가격은 그대로 둔 채 코딩·장기 에이전트 벤치마크를 끌어올렸고, 핵심으로 내세운 건 '정직성'입니다. 같은 날 클로드 코드는 스스로 수십~수백 개 에이전트를 지휘하는 workflow를 공개했고요. 공식 문서와 커뮤니티 반응을 따라가며, 마지막으로 미토스(Mythos) 현황까지 정리했습니다.
