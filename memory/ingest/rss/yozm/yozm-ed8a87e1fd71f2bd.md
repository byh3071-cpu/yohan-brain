# '미토스'급 '페이블 5' 공개: Opus 4.8의 2배 값을 할 것인가 

6월 9일, 앤트로픽이 페이블 5(Fable 5)와 미토스 5(Mythos 5)를 동시에 공개했습니다. 페이블 5는 앤트로픽이 일반에 푼 모델 가운데 역대 가장 강력하고, 미토스 5는 그 페이블에서 안전장치만 들어낸 쌍둥이입니다. 앤트로픽은 지난 Opus 4.8 공개 당시, 미토스 프리뷰 소식도 함께 전하며 '미토스를 일반에 공개하기 위한 강력한 보안 조치에서 빠르게 성과가 나고 있다'고 밝힌 바 있는데요. 결국 미토스 5는 '프로젝트 글래스윙'을 통해 극소수에게만 공개되었고, 일반 사용자는 미토스 본체가 아니라 '페이블'이란 얼굴로, ‘통제’를 더한 미토스를 만나게 됐습니다.
