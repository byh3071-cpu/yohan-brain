# 스스로 키운 수익 — 다섯 가지 축

## 정의 (1~2문장)

- 상속 없이 큰 수익을 낸 사람에게 공통적으로 보이는 **전문성·시장성·성실성·객단가·재구매** 다섯 축으로 오퍼·1인 기업을 점검하는 스코어카드다.
- “지금 잘 안 풀린다”면 금액이 아니라 **어느 축이 약한지**를 먼저 진단한다.

## Verified (소스 기반)

- 상속 없이 **큰 돈을 번 사람**에게 흔히 보이는 요소: **전문성 · 시장성 · 성실성 · 객단가 · 재구매(재계약)**. [source: self-made-wealth-five-elements]
- “지금 잘 안 풀린다”면 위 다섯을 **스코어카드**로 점검. [source: self-made-wealth-five-elements]
- **전문성:** 대체하기 어려운 역량·포지션. [source: self-made-wealth-five-elements]
- **시장성:** 돈 내고 사려는 수요가 있는지. [source: self-made-wealth-five-elements]
- **성실성:** 약속·루틴·납품을 꾸준히 이행하는지. [source: self-made-wealth-five-elements]
- **객단가:** 거래 한 번당 가치·마진. [source: self-made-wealth-five-elements]
- **재구매:** LTV·반복 매출·구독·리텐션. [source: self-made-wealth-five-elements]
- 1인 기업/오퍼 한 종류를 정해 **5줄 체크리스트**만 작성하는 다음 액션. [source: self-made-wealth-five-elements]
- `message_id` 31 — 리소스+서머리 일괄 푸시 완료. [source: self-made-wealth-five-elements]
